Travel OS · Traveler DNA

Tu perfil de viaje, en un flujo limpio y compartible.

Un MVP estático, mobile first y listo para Safari iPhone. Captura preferencias, calcula afinidades, guarda el progreso en localStorage y genera resultados exportables.

Abrir Traveler DNA Ver repo

Hecho para evolucionar después hacia integraciones con GitHub, Supabase, Firebase y Google Sheets sin cambiar la UI.

Responsive real

Experiencia optimizada para iPhone, con botones grandes, progress bar y animaciones suaves.

Persistencia local

El formulario se recupera automáticamente si el navegador se cierra a mitad del flujo.

Resultados exportables

Genera `traveler-dna.json`, `traveler-summary.html` y `traveler-summary.md` desde el resumen.

Arquitectura extensible

La capa de salida queda desacoplada para enchufar canales futuros sin rediseñar la interfaz.

GitHub Pages servirá este sitio desde `docs/`. Si hay un borrador guardado, Traveler DNA lo reanuda automáticamente dentro de la app.
